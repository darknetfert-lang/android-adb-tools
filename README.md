## 🔍 What this tool actually does

This tool executes ADB commands to:

- Uninstall pre-installed apps (bloatware)
- Disable background services
- Optimize system performance
- Improve user privacy by removing trackers

### Example commands used:
```bash
adb shell pm uninstall --user 0 com.facebook.katana
adb shell pm disable-user --user 0 com.google.android.youtube
```

## ⚙️ Instalación

1. Descarga el archivo `.zip`
2. Extrae el contenido
3. Asegúrate de tener ADB instalado
4. Activa "Depuración USB" en tu Android

---

## ⚡ Uso rápido

1. Conecta tu teléfono  
2. Ejecuta el script (`tool.bat`)  
3. Disfruta
